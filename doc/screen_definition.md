# 営業日報システム 画面定義書

## 画面一覧

| No | 画面名 | パス | アクセス可能ロール |
|----|--------|------|--------------------|
| 1 | ログイン画面 | `/login` | 全員（未認証） |
| 2 | ダッシュボード | `/` | sales / manager |
| 3 | 日報一覧画面 | `/reports` | sales / manager |
| 4 | 日報作成画面 | `/reports/new` | sales |
| 5 | 日報編集画面 | `/reports/:id/edit` | sales（自分の draft のみ） |
| 6 | 日報詳細画面 | `/reports/:id` | sales / manager |
| 7 | 顧客マスタ一覧画面 | `/customers` | sales / manager |
| 8 | 顧客マスタ登録・編集画面 | `/customers/new` `/customers/:id/edit` | manager |
| 9 | ユーザーマスタ一覧画面 | `/users` | manager |
| 10 | ユーザーマスタ登録・編集画面 | `/users/new` `/users/:id/edit` | manager |

---

## 1. ログイン画面

**目的：** ユーザー認証を行う

### 表示項目
| 項目 | 種別 | 備考 |
|------|------|------|
| メールアドレス | テキスト入力 | 必須 |
| パスワード | パスワード入力 | 必須 |
| ログインボタン | ボタン | |

### 操作・アクション
- ログインボタン押下 → 認証成功時はダッシュボードへ遷移
- 認証失敗時はエラーメッセージを同画面に表示

### 画面遷移
```
ログイン成功 → ダッシュボード
```

---

## 2. ダッシュボード

**目的：** ログイン後のトップ画面。自分（または全員）の最近の日報サマリーを表示する

### 表示項目
| 項目 | 備考 |
|------|------|
| ログインユーザー名・ロール | ヘッダー表示 |
| 今日の日報ステータス | 未作成 / 下書き / 提出済み / 確認済み |
| 最近の日報一覧（5件） | 日付・ステータス・担当者名（manager のみ担当者名表示） |
| 日報作成ボタン | sales のみ表示。今日の日報が未作成の場合のみ活性 |

### 操作・アクション
- 日報作成ボタン → 日報作成画面へ
- 日報一覧の行クリック → 日報詳細画面へ
- 「日報一覧を見る」リンク → 日報一覧画面へ

### 画面遷移
```
日報作成ボタン → 日報作成画面
日報行クリック → 日報詳細画面
```

---

## 3. 日報一覧画面

**目的：** 日報の一覧を表示・検索する

### 表示項目
| 項目 | 備考 |
|------|------|
| 検索フィルター（日付範囲） | |
| 検索フィルター（担当者） | manager のみ表示 |
| 検索フィルター（ステータス） | |
| 日報一覧テーブル | 下記カラム参照 |

#### 日報一覧テーブル カラム
| カラム名 | 備考 |
|----------|------|
| 日付 | |
| 担当者名 | manager のみ表示 |
| 訪問顧客数 | VISIT_RECORD の件数 |
| ステータス | バッジ表示（下書き / 提出済み / 確認済み） |
| コメント数 | |

### 操作・アクション
- 検索ボタン → 一覧を絞り込み

### 画面遷移
```
詳細ボタン → 日報詳細画面
編集ボタン → 日報編集画面
日報作成ボタン → 日報作成画面
```

## 4. 日報作成画面

**目的：** 当日の日報を新規作成する

**アクセス制限：** sales ロールのみ。当日の日報が既に存在する場合はアクセス不可

### 表示項目

#### 基本情報セクション
| 項目 | 種別 | 備考 |
|------|------|------|
| 日付 | 表示（変更不可） | 本日日付を自動セット |
| 担当者名 | 表示（変更不可） | ログインユーザー名 |

#### 訪問記録セクション
| 項目 | 種別 | 備考 |
|------|------|------|
| 顧客名 | セレクトボックス | 顧客マスタから選択。必須 |
| 訪問内容 | テキストエリア | 必須 |
| 追加ボタン | ボタン | 行を追加（複数行対応） |
| 削除ボタン | ボタン | 各行に配置 |
| 並び替え | ドラッグ＆ドロップ | sort_order を更新 |

#### Problem / Plan セクション
| 項目 | 種別 | 備考 |
|------|------|------|
| 課題・相談（Problem） | テキストエリア | |
| 明日やること（Plan） | テキストエリア | |

#### アクションボタン
| ボタン名 | 動作 |
|----------|------|
| 下書き保存 | status = draft で保存 |
| 提出する | status = submitted で保存。確認ダイアログを表示 |
| キャンセル | 日報一覧画面へ戻る（未保存の場合は確認ダイアログ） |

### バリデーション
- 訪問記録は1件以上必須
- 各訪問記録の「顧客」と「訪問内容」は必須

### 画面遷移
```
下書き保存 → 日報詳細画面（該当日報）
提出する   → 日報詳細画面（該当日報）
キャンセル → 日報一覧画面
```

---

## 5. 日報編集画面

**目的：** 下書き状態の日報を編集する

**アクセス制限：** sales ロールのみ。自分が作成した draft 状態の日報のみ編集可能

### 表示項目・操作

日報作成画面と同じ構成。既存データが入力済みの状態で表示される。

#### アクションボタン
| ボタン名 | 動作 |
|----------|------|
| 保存（下書き） | status = draft で上書き保存 |
| 提出する | status = submitted で保存。確認ダイアログを表示 |
| キャンセル | 日報詳細画面へ戻る |

### 画面遷移
```
保存（下書き） → 日報詳細画面（該当日報）
提出する      → 日報詳細画面（該当日報）
キャンセル    → 日報詳細画面（該当日報）
```

## 6. 日報詳細画面

**目的：** 日報の内容を閲覧し、上長がコメントを記入する

### 表示項目

#### 基本情報
| 項目 | 備考 |
|------|------|
| 日付 | |
| 担当者名 | |
| ステータス | バッジ表示 |

#### 訪問記録
| 項目 | 備考 |
|------|------|
| No | 訪問順 |
| 顧客名 | |
| 訪問内容 | |

#### Problem / Plan
| 項目 | 備考 |
|------|------|
| 課題・相談（Problem） | テキスト表示 |
| 明日やること（Plan） | テキスト表示 |

#### コメントセクション
| 項目 | 備考 |
|------|------|
| コメント一覧 | コメント者名・日時・内容を時系列で表示 |
| コメント入力欄 | テキストエリア。manager のみ表示 |
| コメント送信ボタン | manager のみ表示 |

#### アクションボタン
| ボタン名 | 表示条件 | 動作 |
|----------|----------|------|
| 編集する | sales・自分の日報・draft 状態のみ | 日報編集画面へ |
| 確認済みにする | manager・submitted 状態のみ | status = reviewed に更新 |
| 戻る | 全員 | 日報一覧画面へ |

### 画面遷移
```
編集するボタン → 日報編集画面
戻るボタン    → 日報一覧画面
```

## 画面遷移図

```
[ログイン画面]
    ↓ 認証成功
[ダッシュボード]
    ├─ 日報作成ボタン ──────────────────→ [日報作成画面]
    │                                           ↓ 保存/提出
    ├─ 日報一覧リンク ──→ [日報一覧画面] ←── [日報詳細画面]
    │                          ├─ 詳細 ──→ [日報詳細画面]
    │                          │               ├─ 編集 ──→ [日報編集画面]
    │                          │               │               ↓ 保存/提出
    │                          │               └───────────────┘
    │                          └─ 作成 ──→ [日報作成画面]
    │
    ├─ 顧客マスタ ──→ [顧客マスタ一覧画面]
    │                    ├─ 新規 ──→ [顧客マスタ登録画面]
    │                    └─ 編集 ──→ [顧客マスタ編集画面]
    │
    └─ ユーザー管理（manager のみ）
         └─→ [ユーザーマスタ一覧画面]
                ├─ 新規 ──→ [ユーザーマスタ登録画面]
                └─ 編集 ──→ [ユーザーマスタ編集画面]
```
